# Project Rules

## Stack
- Vite + React + TypeScript
- Tailwind CSS
- Static SPA only

## Data Policy
- Question and answer data must be derived from official RETIO past exam PDFs only.
- Do not add AI-generated legal explanations into the official-only drill flow.
- User notes may be stored locally in `localStorage`.

## Safety
- Do not commit `.env`, keys, credentials, local OCR output, or downloaded PDF artifacts.
