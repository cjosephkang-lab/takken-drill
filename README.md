# 宅建過去問ドリル

直近5回分の宅地建物取引士資格試験の公式過去問を使う、スマホ向けの個人用ドリルです。

## 根拠データ

問題・正解番号は、一般財団法人 不動産適正取引推進機構が公開している「宅建試験の問題及び正解番号表」から取得しています。

- 令和7年度
- 令和6年度
- 令和5年度
- 令和4年度
- 令和3年度12月試験

令和4年度・令和3年度12月試験はPDFがスキャン寄りのため、OCR抽出の不完全箇所があります。各問題カードから公式PDFを開いて原本確認できます。

## 解答解説

各問題には、公式PDFの正解番号表に基づく「解答解説（公式根拠）」を表示します。詳細な法律理由は、公開されている宅建レトスの該当解説ページへリンクします。第三者の解説本文は転載せず、出典ページを直接開く設計です。

あわせて、問題の分野・論点に基づく短い「AI補足メモ」を表示します。これは外部解説の改変転載ではなく、復習時に確認する観点を示すための独自メモです。

## 学習導線（おすすめ順）と合格作戦ボード

効率よく一発合格するための定番セオリーをアプリに組み込んでいます。

- **おすすめ順モード**：ヘッダーの「おすすめ順」をONにすると、合格者が推奨する科目順（最大の得点源である宅建業法 → 最も時間のかかる権利関係 → 法令上の制限 → 税・価格評定 → 免除科目）で出題されます。「次へ」で科目ごとに通しで演習できます。
- **合格作戦ボード**：科目別の正答率を本番1回（50問）に換算した「想定得点」を表示し、目標点・安全圏（38点）まであと何点かを可視化します。

学習導線の科目順・目標配点・合格ラインは、以下の予備校／専門サイトを横断して一貫して支持されている定番セオリーに基づいています（解説本文は転載せず、各社の代表値を採用）。

- 伊藤塾「宅建の合格点」 https://column.itojuku.co.jp/takken/basic/goukakuten/ （合格点：過去10年で最低33点〜最高38点・平均35.5点）
- 伊藤塾「勉強する順番」 https://column.itojuku.co.jp/takken/method/benkyou-junban/
- アガルート「科目構成」 https://www.agaroot.jp/takken/column/composition/
- 全日埼玉「勉強の順番」 https://saitama.zennichi.or.jp/column/study-order/
- 四谷学院「科目別の目標点数」 https://yotsuyagakuin-tsushin.com/blog_takkenshiken/kamokutensuu/

科目順・目標点の具体値とその根拠コメントは `src/data/studyGuide.ts` に集約しています。

## 使い方

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## データ再生成

`source-pdfs/` に公式PDFを置いた上で実行します。

```bash
npm run data
```
