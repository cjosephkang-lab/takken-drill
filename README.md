# 宅建過去問ドリル

直近5回分の宅地建物取引士資格試験の公式過去問を使う、スマホ向けの個人用ドリルです。

## 根拠データ

問題・正解番号は、一般財団法人 不動産適正取引推進機構が公開している「宅建試験の問題及び正解番号表」から取得しています。

- 令和7年度
- 令和6年度
- 令和5年度
- 令和4年度
- 令和3年度12月試験

令和4年度・令和3年度12月試験はPDFがスキャン寄りのため、OCR抽出の不完全箇所があります。各問題カードから公式PDFを開いて原本確認できます。

## 解答解説

各問題には、公式PDFの正解番号表に基づく「解答解説（公式根拠）」を表示します。詳細な法律理由は、公開されている宅建レトスの該当解説ページへリンクします。第三者の解説本文は転載せず、出典ページを直接開く設計です。

## 使い方

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## データ再生成

`source-pdfs/` に公式PDFを置いた上で実行します。

```bash
npm run data
```
